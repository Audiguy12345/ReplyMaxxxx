# ReplyMax build notes

## Goal
Ship a one-screen revenue MVP that generates better cold outreach for founders, agencies, freelancers, and sales operators.

## Constraints
- No auth
- No database
- No billing yet
- No saved history
- No multi-page app
- No platform drift

## Success condition
A user can:
1. enter audience
2. enter offer
3. select platform and tone
4. generate usable outreach in under 15 seconds

## Required output
- 3 openers
- 2 follow-ups
- 3 objection replies
- 1 positioning angle
- 1 CTA recommendation

## Design direction
- dark UI
- clean
- operator-grade
- not playful
- not startup-gimmicky

## Product direction
This is not an “AI writing assistant.”
This is a reply-rate improvement tool.

## Next features only after real usage
1. copy buttons
2. usage cap
3. Stripe paywall
4. saved prompt presets
5. outcome tracking
6. A/B variant mode
